# Credit-Card-Launch-Analysis
Analyzed customer, credit, and transaction data to evaluate the impact of a credit card launch. Identified spending trends and high-value customer segments using Python and SQL. Built insights to power an interactive Streamlit dashboard for business decision-making.

##  Project Overview
This project analyzes customer behavior, transactions, and credit profiles to evaluate the impact of a **credit card product launch / marketing campaign**.  
The final goal is to **design an interactive Streamlit dashboard** that provides business stakeholders with actionable insights into customer engagement, spending patterns, and campaign performance.

---

##  Objectives
- Analyze **customer demographics and credit profiles**
- Understand **transaction behavior before and after the campaign**
- Measure the **impact of the credit card launch** on spending and usage
- Identify **high-value customer segments**
- Build a **Streamlit dashboard** for interactive exploration of insights

---

##  Dataset Description
The project uses multiple structured datasets:

| File Name | Description |
|---------|------------|
| `customers.csv` | Customer demographic information |
| `credit_profiles.csv` | Credit score, credit limit, and risk-related features |
| `transactions.csv` | Transaction-level data (amount, category, time) |
| `avg_transactions_after_campaign.csv` | Aggregated metrics post campaign |
| `E_MasterCardDump.sql` | SQL dump representing raw transactional tables |
